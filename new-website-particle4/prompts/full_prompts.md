# チームシャイニー - 各セクション別プロンプト案

## 1. ヘッダーセクション

### HTMLプロンプト
```
以下の要件を満たすヘッダーセクションのHTMLコードを作成してください：

1. セマンティックなheader要素を使用
2. 左側にロゴ（h1要素でテキスト「チームシャイニー」）
3. 右側に以下のナビゲーションリンク：
   - 選ばれる理由（#features）
   - コース紹介（#courses）
   - 利用者の声（#testimonials）
   - ドロップダウンメニュー「その他」（サポート体制、料金と流れ、よくある質問、お知らせを含む）
   - お問い合わせボタン（#contact、cta-buttonクラス付き）
4. 電話番号表示（Font Awesome電話アイコン付き）
5. モバイル用のハンバーガーメニュートグルボタン（3本線のデザイン）
6. オーバーレイ要素（モバイルメニュー表示時の背景）

特に：
- WAI-ARIA属性を適切に使用してアクセシビリティに配慮する
- Font Awesome 6を使用したアイコン
- シンプルで軽量な構造
```

### CSSプロンプト
```
先ほど作成したヘッダーセクションに対するCSSコードを作成してください：

1. 全体的なスタイリング：
   - ヘッダーは画面上部に固定
   - 白背景、下部に軽いシャドウ
   - 適切なパディングで要素を配置

2. ロゴスタイル：
   - フォント：Poppins（太字）と日本語フォント
   - 適切なフォントサイズとウェイト
   - ブランドカラーを使用

3. ナビゲーションメニュー：
   - 水平配置（Flexboxを使用）
   - 項目間に適切な余白
   - ホバー時のアニメーション効果
   - ドロップダウンメニューのスタイリング

4. 電話番号表示：
   - アイコンと電話番号の適切な配置
   - 見やすいフォントサイズと色

5. ハンバーガーメニュー：
   - モバイル表示時のみ表示
   - 3本線のアニメーション（クリックでX印に変化）

6. レスポンシブデザイン：
   - 1024px以下でハンバーガーメニューに切り替え
   - モバイル時のメニュー展開アニメーション
   - オーバーレイ表示のスタイル

カラーパレットはシンプルで洗練された配色を使用：
- プライマリーカラー：#4a6cf7（青系）
- アクセントカラー：#6c5ce7（紫系）
- テキスト：#333333（濃いグレー）
- 背景：#ffffff（白）
```

## 2. ヒーローセクション

### HTMLプロンプト
```
以下の要件を満たすヒーローセクションのHTMLコードを作成してください：

1. セマンティックなsection要素（heroクラス付き）
2. 中央配置のコンテンツコンテナ
3. 以下の要素を含む：
   - メインキャッチコピー（h2要素）：「発達の特性を活かし、データサイエンス・AI・Webマーケティングのプロへ」
   - サブキャッチコピー（p要素、hero-subtitleクラス付き）：「現役エンジニア・マーケターによる実践指導<br>95%以上が無料利用可能」
   - 2つのCTAボタン：
     a. 「無料相談はこちら」（primary-buttonクラス、#contactリンク）
     b. 「見学・体験を申し込む」（secondary-buttonクラス、#contactリンク）

特に：
- テキストが複数行にわたる場合の適切な改行
- ボタンには適切なクラス名を付与
- シンプルで訴求力のある構造
```

### CSSプロンプト
```
先ほど作成したヒーローセクションに対するCSSコードを作成してください：

1. 全体的なスタイリング：
   - 背景：グラデーション（青→紫）または単色背景
   - 十分な高さ（画面高さの少なくとも80%）
   - 上下のパディングを適切に設定

2. コンテンツ配置：
   - Flexboxを使用した垂直方向の中央配置
   - テキストは中央揃え
   - モバイル表示時も読みやすい配置

3. テキストスタイル：
   - メインキャッチコピー：大きく太字、明るい色（白など）
   - サブキャッチコピー：適度なサイズ、やや薄い色

4. ボタンスタイル：
   - プライマリーボタン：目立つ色、太字
   - セカンダリーボタン：透過背景、ボーダー付き
   - 両方のボタンにホバーエフェクト
   - ボタン間に適切な余白

5. レスポンシブデザイン：
   - 小さい画面ではテキストサイズを調整
   - モバイル表示ではボタンを縦に配置

特に：
- アニメーションはシンプルに保つ
- すべての要素が視認性高く配置
- ヒーローセクションがウェブサイトの顔となるよう印象的なデザイン
```

## 3. 選ばれる理由セクション

```

   a. 専門性の高いコース
      - アイコン：fa-laptop-code
      - 説明：データサイエンス、生成AI、Webマーケティングの専門コースを提供。あなたの興味に合わせて選べます。
   
   b. 現役エンジニアによる指導
      - アイコン：fa-user-tie
      - 説明：業界の最前線で活躍する現役エンジニアが、あなたの成長を丁寧にサポート。実践的な知識とスキルを直接学べます。
   
   c. 実践的な学習内容
      - アイコン：fa-tasks
      - 説明：座学だけではなく、実際のビジネスシーンを想定した課題演習や、就職活動に役立つポートフォリオ作成に力を入れています。

      - アイコン：fa-puzzle-piece

      - 説明：データサイエンスの専門性をさらに高めたい方へ。チームシャイニー卒業後の活躍の場として、Shiny Labをご用意しています。
   
   f. 費用負担の軽減
      - アイコン：fa-yen-sign
      - 説明：経済的な負担を心配することなく、スキルアップに集中できる環境。95%以上の方が無料で利用されています。
   
   g. Webマーケティングも学べる
      - アイコン：fa-chart-line
      - 説明：データ分析、AIに加え、Webマーケティングのスキルも習得可能。多様なキャリアパスを支援します。

特に：
- Font Awesome 6のアイコンを各カードの先頭に配置
- 各カードには適切な余白とスタイルを適用
- グリッドレイアウトで表示（デスクトップでは3列、タブレットで2列、モバイルで1列を想定）
```

## 4. コース紹介セクション

### HTMLプロンプト
```
以下の要件を満たす「コース紹介」セクションのHTMLコードを作成してください：

1. セマンティックなsection要素（id="courses"とsectionクラス付き）
2. 中央配置のセクションタイトル（h2要素、section-titleクラス付き）：「コース紹介」
3. 3つのコースをカード形式で表示（courses-containerクラス内のcourse-cardクラス）：
   
   a. データサイエンスコース
      - アイコン：fa-database
      - 見出し：データサイエンスコース
      - 説明：企業のデータ活用を学び、データ分析のプロを目指す。SQL、Python、Tableauなどを習得。
      - 学習ステップ（リスト）：
        * データ基礎：データの種類と扱い方
        * データ前処理：データのクリーニングと整形
        * データ可視化：効果的なグラフ作成
        * 探索的データ分析：データから洞察を得る
        * 機械学習入門：予測モデルの作成
        * 応用とプロジェクト：実践的なデータ分析
      - 期間：12〜20週間
      - ボタン：「詳しく見る」（#contactリンク）
   
   b. 生成AIコース
      - アイコン：fa-robot
      - 見出し：生成AIコース
      - 説明：画像生成、文章作成、業務効率化など、AIの最先端技術を習得。ChatGPT、Midjourneyなどを活用。
      - 主要ゼミ（リスト）：
        * 生成AIゼミ：画像・テキスト生成の基礎
        * プロンプトエンジニアゼミ：効果的な指示の出し方
        * データ分析（AI活用）ゼミ：AIを活用したデータ分析
        * AI開発ゼミ：AIモデルの開発と実装
        * Excel/スプレッドシートのAI活用ゼミ：業務効率化
      - 期間：8〜16週間
      - ボタン：「詳しく見る」（#contactリンク）
   
   c. Webマーケティングコース
      - アイコン：fa-bullhorn
      - 見出し：Webマーケティングコース
      - 説明：Webサイトへの集客、顧客との関係構築、売上向上を目指す。SEO、SNS、広告運用などを網羅的に学習。
      - 学習内容（リスト）：
        * SEO：検索エンジン最適化の基礎と実践
        * コンテンツマーケティング：効果的なコンテンツ作成
        * SNSマーケティング：各プラットフォームの活用
        * Web広告：Google広告、SNS広告の運用
        * データ分析：アクセス解析と効果測定
      - 期間：10〜18週間
      - ボタン：「詳しく見る」（#contactリンク）

特に：
- 各コースカードは独立した要素として作成
- カード内の要素は適切に構造化（見出し、説明、リスト、フッター情報）
- Font Awesome 6のアイコンを効果的に使用
- チェックマークアイコン（fa-check-circle）をリストアイテム前に配置
- 時計アイコン（fa-clock）を期間表示に使用
```
