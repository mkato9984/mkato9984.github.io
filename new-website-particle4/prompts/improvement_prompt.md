# チームシャイニー ウェブサイト改善プロンプト

```
以下の要件を満たす、発達障害をお持ちの方々向けの「チームシャイニー」ウェブサイトの改善版コードを作成してください。アクセシビリティに配慮した、洗練されたデザインを実現してください。

1. HTML（index.html）：
   - セマンティックなHTML5構造（適切な見出しレベル、ARIA属性の活用）
   - SVGロゴまたは高品質な画像素材の活用
   - ユーザー体験に配慮した情報の階層化
   - スクリーンリーダー対応の隠しテキスト
   - 実際の受講生の写真や活動風景を適切に配置

2. CSS（styles.css）：
   - 一貫性のあるデザインシステムの確立
   - 明確なカラーパレットとコントラスト比
   - アニメーションは控えめで目的を持ったものに限定
   - フォーカス状態の明示的な視覚的フィードバック
   - 発達障害に配慮したテキストの読みやすさ（行間、文字間隔、フォントサイズ）
   - 重要な情報の強調とノイズの削減
   - プレローダーとスケルトンスクリーンの実装

3. JavaScript（main.js）：
   - パフォーマンス最適化（遅延読み込み、コード分割）
   - スムーズなページ内ナビゲーション
   - フォーム入力の即時バリデーション
   - ユーザーインタラクションの視覚的フィードバック
   - エラー状態の明確な伝達

デザインとユーザー体験の強化：
- プロフェッショナルな印象を与える高品質な画像とグラフィック
- 受講生の成功事例を視覚的に魅力的に表現
- 数値データの視覚化（グラフ、チャート）
- 段階的なアニメーションで情報を整理して提示
- ダークモード／ライトモード切り替え機能

コンテンツの強化：
- 実際の受講生の声を引用したテスティモニアル（写真付き）
- 企業との連携や就職実績を具体的に提示
- 学習カリキュラムの視覚的な説明（タイムライン、マイルストーン）
- よくある質問の拡充と検索機能
- 講師陣の実績と専門性の紹介

アクセシビリティの重視：
- WCAG 2.1 AAレベル準拠のデザイン
- キーボードナビゲーションの最適化
- 読み上げソフト対応のマークアップ
- テキストサイズ変更機能
- コントラスト調整機能

レスポンシブデザイン：
- モバイルファーストアプローチ
- 複雑なレイアウトのモバイル最適化
- タッチインタラクションの最適化
- 画面サイズに応じた情報密度の調整

パフォーマンス最適化：
- 画像の最適化と次世代フォーマット（WebP）の利用
- クリティカルCSSの抽出と初期表示の高速化
- 非同期読み込みとレイジーローディング
- サービスワーカーを活用したオフライン対応

カラーパレット：
- プライマリーカラー：#4568DC（アクセシブルな青）
- セカンダリーカラー：#B06AB3（落ち着いた紫）
- アクセントカラー：#FFD166（温かみのある黄色）
- テキストカラー：#333333（読みやすい黒）
- 背景色：#FFFFFF（白）と#F8F9FA（薄いグレー）
- セマンティックカラー：#28A745（成功）、#DC3545（エラー）

タイポグラフィ：
- 見出し：Poppins, bold（英字）と游ゴシック体（日本語）
- 本文：Inter（英字）とNoto Sans JP（日本語）
- 行間：1.6〜1.8で統一
- 文字間隔：読みやすさを確保（特に日本語テキスト）

新規セクションの提案：
1. 「学習の流れ」ビジュアルタイムライン
2. 「卒業生の声と実績」写真とストーリー
3. 「講師紹介」専門性とキャリアパス
4. 「学習環境」教室や設備の紹介
5. 「カリキュラムの詳細」インタラクティブな表示

特に配慮すべき点：
- 視覚的に圧倒されないクリーンなデザイン
- 情報の整理と段階的な提示
- 明確なアクションステップとCTA
- 一貫したナビゲーションとユーザーフィードバック
- 発達障害特性に応じた視覚的・聴覚的配慮
```
